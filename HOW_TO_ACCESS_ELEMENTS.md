## How to Use (Step-by-Step)

### In Main Web Builder:

1. **Open Web Builder**
   - Navigate to `/web-builder` or click "Web Builder" from your dashboard

2. **Switch to Elements Tab**
   - Look at the left sidebar
   - You'll see two tabs: `Navigation` and `Elements`
   - Click **"Elements"**

3. **Browse Components**
   - Use the search bar to find specific elements
   - Or browse by category (Navigation, Sections, Social, etc.)
   - Click category names to expand/collapse

4. **Drag & Drop**
   - Click and hold any element
   - Drag it to the **preview area** (center of screen)
   - You'll see a blue/purple gradient line showing where it will be inserted
   - Release to drop

5. **See Your Element**
   - The element appears in the preview with a smooth animation
   - Hover over it to see controls (Edit, Duplicate, Delete)
   - The HTML code is automatically added to your project

6. **Export or Save**
   - Click "Export" in the toolbar to download HTML
   - Or click "Save" to store your work

## Troubleshooting

### "I don't see the Elements tab"
- Make sure you're on `/web-builder` (not `/elements-builder`)
- Check the left sidebar at the top - it should show two tabs
- If the left panel is collapsed, click the chevron button to expand it

### "Elements won't drag"
- Make sure you're clicking and holding the element card
- Look for the drag cursor (hand icon)
- Try refreshing the page if drag doesn't work

### "Drop doesn't work"
- Drag the element over the **white preview area** (center screen)
- Look for the blue gradient line showing insertion point
- Make sure you're in "Canvas" view mode (not "Code")

### "I want the standalone version"
- Navigate to `/elements-builder` for the dedicated demo page
- This has a cleaner interface focused only on drag-and-drop
