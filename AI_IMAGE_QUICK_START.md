## ğŸ”§ Troubleshooting

### Image generation is slow
- **Solution**: Use "Standard" quality for faster results
- **Tip**: "High" quality takes 15-25 seconds (normal)

### Image doesn't match my prompt
- **Solution**: Be more specific and descriptive
- **Tip**: Add style keywords like "professional", "modern", "vibrant"

### Can't insert to canvas
- **Check**: Make sure Web Builder is active
- **Try**: Refresh page and generate again

### Image quality is poor
- **Solution**: Use "High" or "Ultra" quality
- **Tip**: Add "high quality, detailed, 8k" to prompt
